# TripCart Context Load

이 프로젝트의 정본 문서를 순서대로 읽고 현재 상태를 요약하라.

## 읽기 순서
1. `docs/00_READ_THIS_FIRST.md`
2. `docs/PRODUCT_MASTER_PLAN.md`
3. `docs/ARCHITECTURE.md`
4. `docs/API_CONTRACT_v0.2.md`
5. `docs/tripcart_schema_canonical_v0.3.sql` (앞부분 구조만)
6. `docs/DESIGN_SYSTEM.md`
7. `docs/GOVERNANCE.md`

## 출력 형식
1. 프로젝트 현재 상태 요약 (3~5문장)
2. 가장 최근 변경된 문서
3. 현재 품질 점수 Band
4. 다음 우선순위 작업 (ROADMAP 기준)
5. 열린 리스크 또는 미해결 사항
